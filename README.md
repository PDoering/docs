## Installation

See the official [documentation](https://www.mkdocs.org/#installation).

## Commands

* `mkdocs serve` - Start the live-reloading docs server.
* `mkdocs gh-deploy` - Deploy pages to GitHub.
* `mkdocs help` - Print help message.
